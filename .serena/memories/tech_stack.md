# 기술 스택

## 프론트엔드
- **프레임워크**: React 19.2.0
- **빌드 도구**: Vite 7.2.4
- **언어**: JavaScript (ES2020+, JSX)
- **상태 관리**: Zustand 5.0.8
- **HTTP 클라이언트**: Axios 1.7.2
- **개발 도구**:
  - ESLint 9.39.1
  - @vitejs/plugin-react 5.1.1
  - netlify-cli 23.12.2

## 백엔드
- **프레임워크**: FastAPI
- **서버**: Uvicorn
- **데이터베이스**: PostgreSQL
- **ORM**: SQLAlchemy 2.0+
- **인증**: PyJWT 2.0.0+, passlib[bcrypt]
- **환경 변수**: python-dotenv
- **드라이버**: psycopg2-binary

## 인프라
- **컨테이너화**: Docker, Docker Compose
- **웹 서버**: Nginx
- **IaC**: Terraform
- **CI/CD**: Google Cloud Build

## 개발 환경
- **운영체제**: macOS (Darwin)
- **버전 관리**: Git
