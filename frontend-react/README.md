# Energy Tycoon - React Frontend

React + Vite + Zustand로 구현된 Energy Tycoon 게임의 프론트엔드입니다.

## 🚀 시작하기

### 1. 백엔드 서버 실행 (필수)

먼저 백엔드 서버를 실행해야 합니다:

\`\`\`bash
# 프로젝트 루트 디렉토리에서
cd ..
python main.py
\`\`\`

백엔드 서버가 http://localhost:8000 에서 실행됩니다.

### 2. 프론트엔드 개발 서버 실행

\`\`\`bash
npm run dev
\`\`\`

브라우저에서 http://localhost:5173 을 열면 게임을 시작할 수 있습니다.

## 🎮 게임 사용법

### 발전기 설치
1. 하단의 "발전기" 탭 클릭
2. 원하는 발전기를 드래그
3. 메인 화면에 드롭하여 설치

### 발전기 관리  
- 클릭하여 상세 정보 확인
- 운영 중단/재개, 업그레이드, 철거 가능

### 화면 이동
- 마우스 휠 또는 화살표 키로 좌우 이동

## 🐛 문제 해결

백엔드가 http://localhost:8000 에서 실행 중인지 확인하세요!
